a11y-check.yml - це конфігураційний файл для GitHub Actions, який автоматично запускає тести доступності (a11y) за допомогою `pa11y-ci` при кожному пуші або створенні пул-реквеста в гілку `master`. Цей workflow вилучили із загального процесу CI/CD, щоб зосередитися виключно на тестуванні доступності з допомогою e2e тестування.
